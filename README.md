# gredle-sample
This is a sample for usage of gradle. <br>
You can learn more from http://spring.io/guides/gs/gradle/

## How to run ?
$ git clone https://github.com/nfuwang/gradle-sample && cd gradle-sample <br>
$ ./gradlew run <br>
